# CookOld

Минимальный iOS-проект (iOS 18)

## Работа с Git

### Хуки
Склонировав проект в первый раз, необходимо ввести в терминале следующую команду:
```shell
make hook
```
Эта команда установит стандартный путь до гит-хуков `.githooks`.

> Теперь, при каждом чекауте или мерже, автоматически будет будет запускаться команда `make xcgen`.

## Установка зависимостей

### Homebrew
Чтобы установить dev-зависимости проекта, установи [Homebrew](https://brew.sh/).

### Mint
Затем установи менеджер зависимостей [Mint](https://github.com/yonaskolb/Mint):

```shell
brew install mint
```

После установки Mint:

```shell
mint bootstrap
```

Эта команда установит зависимости из `Mintfile` (в том числе **XcodeGen**).

## Генерация проекта с последующим открытием

```bash
make generate open
```
